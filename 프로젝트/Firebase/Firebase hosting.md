# 파이어베이스 프로젝트 만들기

# Firebase Hosting
* 파이어베이스 설치
`npm install -g firebase-tools`

* 구글 로그인

`firebase login`

* 프로젝트 초기화
`firebase init`

* 프로젝트 배포
`firebase deploy`

* 로컬에서 실행
`firebase serve`

* 이외 커맨드
`login, list, use, open, help`

* 롤백
파이어베이스 콘솔에서 가능
