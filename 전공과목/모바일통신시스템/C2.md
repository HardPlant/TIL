# 전파

* 인공적 매개물 없이 공간에 전달하는 3kGHz보다 낮은 주파수 전자파

* 발견 : 맥스웰 방정식(전파가 있을 가능성) -> 헤르츠의 실험 (전파의 포착)

* 으용 : 무선전신기 (마르코니)

## Transverse EletroMagnetic

* 전기장, 자기장, 파형의 진행 방향이 상호 직각

## 전파의 발생

* 직류 : 시간에 따른 전하, 전기력선의 변화
