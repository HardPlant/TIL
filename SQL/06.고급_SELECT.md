# 고급 Select

```sql
CREATE TABLE movie_table(
    movie_id INT NOT NULL AUTO_INCREMENT,
    title VARCHAR(150),
    rating VARCHAR(2),
    drama BOOLEAN,
    comedy BOOLEAN,
    action BOOLEAN,
    gore BOOLEAN,
    scifi BOOLEAN,
    for_kids BOOLEAN,
    cartoon BOOLEAN,
    purchased DATE,
    PRIMARY KEY (movie_id)
);
```

* 문제점

  * 영화 카테고리를 알 수 없음 (반납 시 애로사항)

  * 영화가 어떤 내용인지 알기 힘음 (장르 중복 가능)

  * 참, 거짓 데이터 채우기 힘듬 (모든 T/F 열을 채워야 함)

* 카테고리 열 만들기

```sql
ALTER TABLE movie_table
ADD COLUMN category VARCHAR(100);
```

* 열 채우기

```sql
UPDATE movie_table
SET category = 'drama'
where drama = 'T';
UPDATE movie_table
SET category = 'comedy'
where comedy = 'T';
--................--
UPDATE movie_table
SET category = 'family'
where cartoon = 'T' AND RATING = 'G';
UPDATE movie_table
SET category = 'ㅡㅑㄴㅊ'
where cartoon = 'T' AND RATING <> 'G';
```

* 더 나은 방법 : CASE

```sql
UPDATE movie_table
SET category =
CASE
    WHEN drama = 'T' THEN 'drama'
    WHEN comedy = 'T' THEN 'comedy'
    WHEN cartoon = 'T' AND rating = 'G' THEN 'family'
    ELSE 'misc'
END;
```

```sql
UPDATE movie_table
SET category =
CASE
    WHEN category = 'drama' AND rating='R' THEN 'drama-r'
    WHEN category = 'horror' AND rating='R' THEN 'horror-r'
    WHEN category = 'misc' AND rating='G' THEN 'family'
END;
```

```sql
UPDATE movie_table
SET category =
CASE
    WHEN RIGHT(category, 2) = '-r' AND rating='R' THEN SUBSTRING_INDEX('-')
END;
```

```sql
UPDATE movie_table
DROP COLUMN drama
END;
```

* ORDER BY

```sql
SELECT title, category FROM movie_table WHERE title LIKE 'A%' AND category = 'family';
SELECT title, category FROM movie_table WHERE title LIKE 'B%' AND category = 'family';
SELECT title, category FROM movie_table WHERE title LIKE 'C%' AND category = 'family';
```

```sql
SELECT title, category FROM movie_table
WHERE category = 'family'
ORDER BY title;
```

* 열을 모두 더한 값 출력

```sql
SELECT first_name, SUM(sales)
FROM cookie_sales GROUP BY first_name
ORDER BY SUM(sales);
```

* 하루 평균이 가장 높은 값

```sql
SELECT first_name, AVG(sales)
FROM cookie_sales
GROUP BY first_name;
```

* MIN, MAX

```sql
SELECT first_name, AVG(sales)
FROM cookie_sales
GROUP BY first_name;
```

* 날 세기

```sql
SELECT first_name,COUNT(sale_date)
FROM cookie_sales
GROUP BY first_name;
```

* 중복 제거

```sql
SELECT DISTINCT sale_date
FROM cookie_sales
ORDER BY sale_date;
```

```sql
SELECT COUNT(DISTINCT sale_date)
FROM cookie_sales
ORDER BY sale_date;
```
