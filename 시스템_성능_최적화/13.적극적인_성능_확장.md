# 적극적인 성능 확장

성능 튜닝의 한 축은 서비스 간, 서비스 내에서 반복되는 로직을 제거하는 것이다.
결과를 재사용하면 반복되는 로직을 제거할 수 있다.

* 시스템 구성 요소 캐시 사용 예

브라우저:
    * 도메인명 주소
    * 콘텐츠
WAS:
    * 환경설정 개키
    * 프로그램 모듈 캐시
프레임워크:
    * 환경설정
    * 서비스, 프로그램
    * 송수신 전문 구조
    * SQL
    * 애플리케이션 캐시 제공
DBMS:
    * 환경설정
    * 테이블, 인덱스, 권한 등 DB 구성정보
    * SQL 실행계획
    * 데이터 캐시
애플리케이션:
    * 적극적인 사용 요구
