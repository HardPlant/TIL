#Node-RED
https://developer.artik.io/documentation/artik/tutorials/node-red.html

* 드래그 앤 드롭 프로그래밍이 가능해진다.

1. 초기 설정
* 와이파이를 설정한다.
* NodeJS를 설치한다.
```
dnf upgrade 
dnf install node 
dnf install npm 
```

* `Node-RED`를 설치한다.
```
npm -g install node-red 
npm -g install node-red-contrib-artik 
npm -g install node-red-contrib-artik-cloud 
```
