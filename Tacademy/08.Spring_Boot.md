### Code

##### @SpringBootApplication

```java
@SpringBootApplication
public class BootSpringBootApplication{
public static void main(String[] args ){
    SpringApplication.run(BootSpringBootApplication.class, args);
}
}
```

자바 App 진입엄 psvm 메서드 안에 인슨턴스를 생성해 run을 실행함
