# matplotlib

```py
import matplotlib.pyplot as plt
import numpy as np
```
### 데이터 생성

```py
x = np.linspace(0, 5, 11)
y = x ** 2
```

### 그래프 그리기

```py
# Functional Method
plt.plot(x,y)
plt.xlabel('X Label')
plt.ylabel('Y Label')
plt.title('Title')
```

서브플롯

```py
plt.subplot(1,2,1)
plt.plot(x,y)

plt.subplot(1,2,2)
plt.plot(y,x)
```
