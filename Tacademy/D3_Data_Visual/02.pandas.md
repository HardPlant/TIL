### Series

`pd.Series([10, 20, 30])`

인덱스를 붙여서 보내줌
인덱스가 부여되므로 서로 다른 시리즈끼리 연산이 가능함

#### 인덱스 부여

```py
ser1.index = ['a','b','c']
```

#### 딕셔너리 -> 시리즈 변화

```py
ser2 = pd.Series({'a':100, 'b':200, 'c':300})
ser2
```

#### 키워드를 이용한 시리즈

```py
ser3 = pd.Series(data=[1,2,3,4], index=['USA','Germany', 'France', 'Japan'])
ser3
```
